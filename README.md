# 🚗 Catálogo Técnico Automotivo Inteligente

## 📝 Objetivo do Projeto
Este projeto é uma aplicação web simples desenvolvida para a disciplina de Inteligência Artificial. O sistema funciona como um catálogo automotivo inteligente: o usuário informa o modelo, versão e ano de um veículo, e a aplicação se conecta à API do OpenRouter para gerar uma ficha técnica completa, incluindo motorização, consumo estimado, itens de série, além de pontos fortes e fracos do modelo.
